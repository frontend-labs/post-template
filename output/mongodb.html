¿Cómo instalar MongoDB Community Edition?

Hay ocasiones en las que necesitamos habilitar o deshabilitar una caja de texto, un botón ó cualquier otro elemento de un formulario html con jQuery.

Para llevar esto a cabo usaremos el método jQuery llamado .prop()

.prop( nombreDeUnaPropiedad, valor )

Este método obtiene el valor de una propiedad en un elemento html, y desde luego también nos permite establecer un valor para una propiedad.

Parámetro: nombreDeUnaPropiedad
Tipo: String
Descripción: Este parámetro sirve para especificar un nombre de una propiedad.

Parámetro: valor
Tipo: Cualquiera
Descripción: Este parámetro sirve para especificar un valor para la propiedad, aquí también podemos usar una función, siempre y cuando retorne un valor.

Parámetro: propiedadesMultiples
Tipo: Objeto
Descripción: Este parámetro sirve para especificar pares propiedad-valor para una ó más propiedades, mediante un objeto javascript.
